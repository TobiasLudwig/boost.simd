#define BOOST_SIMD_NO_INFINITIES

#include <boost/simd/config.hpp>

Infinites handling behavior option.

By defining this symbol, Boost.SIMD mathematical functions will not take care of handling infinites correctly. This option usually provides increased performance at the cost of correct handling of infinites related corner cases.
